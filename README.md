# Playground

实验性项目集合。

## 项目列表

### appflow-landing

移动效率应用 AppFlow 的 Landing Page。

- **技术栈**: React 18 + TypeScript + Vite + Tailwind CSS
- **设计风格**: 现代简约

```bash
cd appflow-landing
npm install
npm run dev
```

访问 http://localhost:5173 查看效果。

**页面结构**:
- Header - 固定导航栏
- Hero - 主视觉区域，包含下载按钮
- Features - 6 个功能特性卡片
- AppPreview - 手机模型展示应用界面
- CTA - 下载行动号召区
- Footer - 页脚导航和社交链接
